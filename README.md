# 🎬 Movie Rental Analytics Dashboard | Power BI + SQL

## Project Overview

The objective of this project is to create a comprehensive Power BI dashboard using the Sakila DVD Rental Store Database, providing valuable insights into the rental store business. The primary aim is to optimize film inventory, enhance customer satisfaction, improve staff performance, and streamline store operations.

---

## Objectives

* Analyze rental and sales performance
* Understand customer behavior and segmentation
* Track monthly revenue trends
* Evaluate inventory availability
* Identify popular film categories and languages
* Measure store and staff performance
* Generate business insights through visualization

---

## Tools & Technologies

* Power BI
* SQL (PostgreSQL)
